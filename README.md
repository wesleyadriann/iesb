## Rodando
#### Python
```
python3 <file_name>.py
```
#### C
```
./build.sh <file_name>.c <file_name_out>
```
#### Octave
```
octave <file_name>.m
```

<file_name> é o nome do arquivo.  
<file_name_out> é o nome do arquivo após compilado.
